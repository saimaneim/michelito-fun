#  Michelito GIF Generator

Genera mensajes de Discord de Michelito como GIF de 1 frame para guardar con ⭐ en Discord.

## Setup

```bash
# 1. Instalar dependencias
bun install

# 3. Correr
bun dev
# → http://localhost:3000
```

## Uso

Abre `http://localhost:3000`, escribe el mensaje, click en **Generar GIF**.  
El GIF se descarga automáticamente — súbelo a Discord y dale ⭐.

## Objetivos 

- [] Conseguir fuente oficial de discord
- [] Obtener dinamicamente el avatar del usuario
- [] Obtener la etiqueta que usa el usuario
